# My website

## Development

Make sure you have ruby and bundler installed.

Run
```
$ bundle install --path vendor/bundle
```

Serve the site locally
```
$ bundle exec jekyll serve
```

## Update

Keep the site up to date with the GitHub Pages gem:
```
$ bundle update
```
